# AI Agent Rules for Electricity Module
- **Tech Stack:** Next.js 15 (App Router), React, TailwindCSS, TypeScript.
- **Modularity:** NEVER write massive arrays in a single file. If adding data, create a separate JSON or TS file in `src/data/` and import it.
- **Styling:** Use Tailwind utility classes. Follow the existing dark/slate theme.
- **Components:** Always check `components.json` or `src/components/ui/` before creating a new UI component. Reuse existing ones.
